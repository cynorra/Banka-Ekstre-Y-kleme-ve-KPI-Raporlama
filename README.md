# Banka Ekstre Analiz API & Dashboard

Bu proje, kullanıcıların banka ekstrelerini (CSV) sisteme yükleyerek harcamalarını analiz edebilecekleri, otomatik kategorilendirme yapabilen ve finansal KPI raporları sunan bir REST API ve akıllı Dashboard uygulamasıdır.

Backend Django + DRF ile, veritabanı PostgreSQL ile geliştirilmiş olup tüm yapı Docker üzerinde çalışmaktadır.

# Teknolojiler

- Backend: Python 3.10, Django 5, Django REST Framework
- Veritabanı: PostgreSQL 15
- Dokümantasyon: Swagger / OpenAPI (Drf-Yasg)
- Konteyner: Docker & Docker Compose
- Frontend: HTML5, Bootstrap 5, Chart.js (Basit Dashboard)

MMM projesi için geliştirilmiştir.
